# **App Name**: MealPrep Pro Landing Page

## Core Features:

- Responsive Landing Page Display: Render all essential content sections such as the Navbar, Hero, Features, Pricing, Meals Gallery, FAQ, and Footer with full Right-to-Left (RTL) support and adaptive layouts for various screen sizes, including a mobile-specific sticky bottom CTA bar.
- Interactive Theme Toggle: Implement a light/dark mode switching mechanism that respects system preferences, persists the user's choice using localStorage, and utilizes Tailwind CSS's dark mode class strategy for consistent theming.
- Trial & Subscription Form Handling: Provide user-friendly drawers for submitting trial week and monthly subscription requests. This includes capturing user details (name, phone, email, selected plan, and dietary goals) with client-side validation and providing mock submission feedback (success/error toasts).
- Nutrition Consultation Booking Form: Integrate a modal form for users to book a free nutrition consultation. The form will collect personal information (name, phone, email), dietary goals, and preferred consultation time, complete with client-side validation and mock submission confirmation.
- Arabic Content & RTL Layout: Ensure all application text content is in Arabic Egyptian, supporting a friendly and professional tone, with proper Right-to-Left (RTL) text alignment and layout application across all components.
- AI-powered Meal Plan Suggestion Tool: Develop a basic tool that suggests initial meal combinations from the available meal gallery to new users, based on their chosen dietary goals (e.g., weight loss, muscle gain, keto, vegan) provided during subscription signup. This helps fulfill the 'or let us choose' customization option by leveraging basic AI reasoning.

## Style Guidelines:

- Primary brand color: A refined, muted green (#668F3D). This hue evokes feelings of health, nature, and professionalism, setting a calm yet vibrant tone for interactive elements and key brand accents.
- Background color: A very subtle, almost off-white shade with a delicate hint of green (#F2F4F0). This provides a clean, airy, and welcoming canvas that softly complements the primary color, ensuring excellent readability across light themes.
- Accent color: A deeper, richer green (#1F7A1F). This color provides a strong, dynamic contrast, serving as a compelling visual cue for primary call-to-action buttons and essential highlights, effectively drawing the user's eye to critical interactions.
- Headlines will utilize 'Space Grotesk' (sans-serif) to convey a modern, slightly technical, and precise aesthetic that complements the 'Pro' aspect of the brand and highlights the focus on calculated macros. Body text will employ 'Inter' (sans-serif) for its high legibility, neutral, and objective readability, ensuring a seamless and accessible user experience.
- Modern, clean, and minimalistic outline-style SVG icons will be consistently employed throughout the interface. These icons will visually represent micro-benefits, features, and navigation elements, reinforcing the clean, SaaS-like aesthetic.
- The page will feature a structured, single-page layout with distinct sections for clear content hierarchy. A mobile-first approach is paramount, including a responsive sticky bottom CTA bar on mobile, swipe-friendly pricing cards, and an accordion FAQ for optimized user interaction on smaller screens. All elements will naturally support Right-to-Left (RTL) content direction.
- Subtle, lightweight animations will be strategically implemented for smooth transitions and state changes. Examples include elegant section entrances, responsive theme toggle, and accordion expansions, enhancing the overall user experience without causing distraction or sacrificing performance.
